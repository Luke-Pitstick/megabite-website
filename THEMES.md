## ðŸŽ¯ How to Switch Themes

1. Open `tailwind.config.mjs`
2. Find the `colors:` section (around line 6)
3. Replace the `primary` and `secondary` color objects with your chosen theme
4. Save the file
5. Run: `npm run build`
6. Run: `npm run deploy`

**That's it!** Your entire site will update to the new colors.

---

## ðŸ’¡ Pro Tips

- **Primary** = Main brand color (buttons, CTAs, accents)
- **Secondary** = Supporting color (borders, hover states, headings)
- Mix and match! Try primary from one theme + secondary from another
- The logo text will always show as: Primary for "Bite", Secondary for "Mega"
